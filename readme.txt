How to use
Say you find a photo online and the name parses like a facebook file- a bunch of random numbers and underscores. Now say you want to find the individual to whom this photo belongs. Here comes "FaceBook Reverse Photo Finder". Simply input the file name and the program will grab the users profile URL, photo and sex. Here is an example file name to get you started: 33928_152487554786401_100000754713948_237838_7543147_n.jpg
